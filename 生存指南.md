### lldb 的使用

- b 断点
- r 运行直到断点
- s 进入
- n 下一步
- bt 栈
- f 帧
- c 继续执行

可以参考 https://dbgtech.github.io/Tools/lldb-using.html

### objdump

```sh
objdump -d <可执行文件>
```

查看编译器生成的默认构造函数

```sh
objdump -C -t <可执行文件>
```

`-C` 选项告诉 `nm` 命令对 C++ 符号进行 demangling（去花式符号）。这样可以将 C++ 编译器产生的符号名转换成人类可读的形式，以便更容易理解和分析输出结果。
`-t` 选项告诉 `objdump` 命令显示符号表。这将列出目标文件或可执行文件中的所有符号及其地址。
